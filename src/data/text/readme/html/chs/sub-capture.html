  • 使用时机

    游戏UI和对话选项不会自动翻译，需手动选择撷取，建议使用较为精准的Google Vision。

  • 使用方法

    点选 按钮或是按下Ctrl+F11开启【翻译萤幕文字】视窗，然后在撷取视窗中选取欲翻译的文字(如下方影片所示)

  • 功能说明

    1. 左上角的下拉选单可选择撷取方式。
    2. 换行切割：勾选后会以【行】为单位逐行翻译(用来翻译过场的选择项目)，不勾选的话会将所有文字合併成一行再翻译。 (翻译整篇文章时使用)
    3. 编辑文字：勾选后会在撷取文字后显示编辑视窗，可先在此手动修正文字再按【翻译】提交，不勾选的话则会直接提交翻译，若提交的文字有误也不会修正。

  • 撷取方式

    • Google Vision (推荐)

      文字辨识能力精准，几乎不会有错字，需在GCP建立API凭证才能使用。 ( 查看设定教学 )

    • GPT Vision

      性能和Google Vision差不多，如果你是使用GPT翻译即可直接使用。

    • Tesseract OCR

      有时会有错字，需手动编辑修正
