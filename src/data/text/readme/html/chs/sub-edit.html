  • 开启【重新翻译 & 自订翻译】视窗

    点选Tataru Helepr Node的翻译文字

  • 重新翻译

    1. 选择【翻译引擎】和【语言】
    2. 点选【重新翻译】按钮

  • 新增自订翻译

    1. 在【替换前(原文)】输入欲替换的原文
    2. 在【替换后(自订翻译)】输入替换的文字
    3. 点选【类别】选单，选择自订翻译的类型
    4. 点选【储存自订翻译】按钮

  • 删除自订翻译

    1. 在【替换前(原文)】输入欲删除的自订翻译原文
    2. 点选【类别】选单，选择自订翻译的类别
    3. 点选【删除自订翻译】按钮
