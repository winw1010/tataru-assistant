本功能需绑定付款资讯，但每个月有1000张的免费额度，不超过就不会扣款， 点此查看收费方式

使用此功能前需启用Google Cloud Platform功能，请点 此连结 启用


使用API金钥

  1. 前往 Google Cloud Platform 启用Google Cloud Platform服务，资讯可以任意填写，本功能需使用Google帐号，若没有Google帐号请建立一个
  2. 前往 资讯主页 点选【选取专案】，选择一个近期专案

    若无专案请点选【建立专案】建立一个新专案
  3. 前往 帐单帐户管理页面 点选【建立帐户】建立一个帐单帐户，建立后填入付款资料， 并连结上一步骤选取的专案
  4. 至 API和服务页面 ，点选【建立凭证】，然后点选【API金钥】建立金钥
  5. 建立金钥之后点选刚刚建立的API金钥，在【API限制】点选【限制金钥】，然后搜寻并勾选【Cloud Vision API】，设定完毕后按下【储存】即可
  6. 点选刚刚建立的API金钥右侧的【显示金钥】，按下金钥右边複製按钮即可複製起来
  7. 进入Tataru Assistant的【设定】，左上角下拉选单选择【API设定】，然后在Google Vision的【验证方式】栏位选择【API Key】
  8. 在Google Vision的【API Key】栏位输入刚刚複製的API金钥
  9. 在撷取视窗选择Google Vision即可开始使用

使用JSON档案

  1. 前往 Google Cloud Platform 启用Google Cloud Platform服务，资讯可以任意填写，本功能需使用Google帐号，若没有Google帐号请建立一个
  2. 前往 资讯主页 点选【选取专案】，选择一个近期专案

    若无专案请点选【建立专案】建立一个新专案
  3. 前往 帐单帐户管理页面 点选【建立帐户】建立一个帐单帐户，建立后填入付款资料， 并连结上一步骤选取的专案
  4. 至 Google Vision API页面 选择上一步骤选取的专案并启用Google Vision
  5. 前往 API/服务详细资料 ，点选【建立凭证】，然后点选【服务帐户】建立一个服务帐户(资料随意填即可)

  6. 建立后点选上一步建立的服务帐户，点选【金钥】，然后点选【新增金钥】>【建立新的金钥】>【JSON】，然后按下创建后即会自动下载凭证至您的电脑




  7. 进入Tataru Assistant的【设定】，左上角下拉选单选择【API设定】，然后在Google Vision的【验证方式】栏位选择【JSON档案】
  8. 按下【开启Google凭证档案】按钮，开启刚刚下载的JSON档案

  9. 在撷取视窗选择Google Vision即可开始使用
  10. 由于凭证和您的帐单绑定，因此请妥善保管凭证档案，若想移除凭证请至您的服务帐户的金钥页面删除即可
