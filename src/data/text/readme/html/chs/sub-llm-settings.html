回首页

  1. 按下 开启设定，然后在左上角的下拉选单选择【API设定】。
  2. 从AI官网申请API金钥，过程这里就不赘述，取得API金钥之后将其输入到API设定中该AI的【API Key】栏位里。
  3. 若想使用非预设模型，请在该AI的【模型】栏位输入想使用的模型代码，模型代码可从下方连结複製。
    Gemini https://ai.google.dev/gemini-api/docs/models
    GPT https://developers.openai.com/api/docs/models
    Cohere https://docs.cohere.com/docs/models
    Kimi https://platform.kimi.ai/docs/models
  4. 在左上角的下拉选单选择【翻译设定】，在【翻译器】栏位选择你使用的AI名称。
  5. 若想提高AI翻译的精准度，请在左上角的下拉选单选择【LLM设定】，启用【多轮对话】，并将【对话长度】设定在5以上，长度越高精准度就越好，但Token也会增加。
  6. 按下储存按钮储存刚刚的设定即可开始使用AI翻译。
