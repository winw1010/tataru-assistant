  • 拖曳按钮，按住可以移动视窗

  • 开启【设定】视窗 (快捷键Ctrl+F10)

  • 开启【 撷取萤幕文字 】视窗，用来翻译无法自动翻译的文字 (快捷键Ctrl+F11)

  • 开启或关闭【滑鼠穿透】功能，开启后滑鼠可点选Tataru Assistant视窗后的东西

  • 更新提示按钮，在新版本释出时会自动出现

  • 缩小Tataru Assistant视窗

  • 关闭Tataru Assistant视窗

  • 开启或关闭【文字语音】功能，开启后会将文字转换成Google语音并播放

  • 开启【翻译查询】视窗，可作为翻译机使用，若勾选【使用Tataru翻译】则会使用Tataru Assistant来翻译

  • 开启【读取对话纪录】视窗，可查看之前的翻译纪录

  • 删除最后一句

  • 删除全部文字
  • 快捷键
    按键 功能
    Ctrl+F9 使用说明书
    Ctrl+F10 设定
    Ctrl+F11 翻译萤幕文字
    Ctrl+F12 开发者工具
