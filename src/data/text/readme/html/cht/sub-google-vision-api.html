本功能需綁定付款資訊，但每個月有1000張的免費額度，不超過就不會扣款， 點此查看收費方式

使用此功能前需啟用Google Cloud Platform功能，請點 此連結 啟用


使用API金鑰

  1. 前往 Google Cloud Platform 啟用Google Cloud Platform服務，資訊可以任意填寫，本功能需使用Google帳號，若沒有Google帳號請建立一個
  2. 前往 資訊主頁 點選【選取專案】，選擇一個近期專案

    若無專案請點選【建立專案】建立一個新專案
  3. 前往 帳單帳戶管理頁面 點選【建立帳戶】建立一個帳單帳戶，建立後填入付款資料， 並連結上一步驟選取的專案
  4. 至 API和服務頁面 ，點選【建立憑證】，然後點選【API金鑰】建立金鑰
  5. 建立金鑰之後點選剛剛建立的API金鑰，在【API限制】點選【限制金鑰】，然後搜尋並勾選【Cloud Vision API】，設定完畢後按下【儲存】即可
  6. 點選剛剛建立的API金鑰右側的【顯示金鑰】，按下金鑰右邊複製按鈕即可複製起來
  7. 進入Tataru Assistant的【設定】，左上角下拉選單選擇【API設定】，然後在Google Vision的【驗證方式】欄位選擇【API Key】
  8. 在Google Vision的【API Key】欄位輸入剛剛複製的API金鑰
  9. 在擷取視窗選擇Google Vision即可開始使用

使用JSON檔案

  1. 前往 Google Cloud Platform 啟用Google Cloud Platform服務，資訊可以任意填寫，本功能需使用Google帳號，若沒有Google帳號請建立一個
  2. 前往 資訊主頁 點選【選取專案】，選擇一個近期專案

    若無專案請點選【建立專案】建立一個新專案
  3. 前往 帳單帳戶管理頁面 點選【建立帳戶】建立一個帳單帳戶，建立後填入付款資料， 並連結上一步驟選取的專案
  4. 至 Google Vision API頁面 選擇上一步驟選取的專案並啟用Google Vision
  5. 前往 API/服務詳細資料 ，點選【建立憑證】，然後點選【服務帳戶】建立一個服務帳戶(資料隨意填即可)

  6. 建立後點選上一步建立的服務帳戶，點選【金鑰】，然後點選【新增金鑰】>【建立新的金鑰】>【JSON】，然後按下創建後即會自動下載憑證至您的電腦




  7. 進入Tataru Assistant的【設定】，左上角下拉選單選擇【API設定】，然後在Google Vision的【驗證方式】欄位選擇【JSON檔案】
  8. 按下【開啟Google憑證檔案】按鈕，開啟剛剛下載的JSON檔案

  9. 在擷取視窗選擇Google Vision即可開始使用
  10. 由於憑證和您的帳單綁定，因此請妥善保管憑證檔案，若想移除憑證請至您的服務帳戶的金鑰頁面刪除即可
