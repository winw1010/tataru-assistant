回首頁

  1. 按下 開啟設定，然後在左上角的下拉選單選擇【API設定】。
  2. 從AI官網申請API金鑰，過程這裡就不贅述，取得API金鑰之後將其輸入到API設定中該AI的【API Key】欄位裡。
  3. 若想使用非預設模型，請在該AI的【模型】欄位輸入想使用的模型代碼，模型代碼可從下方連結複製。
    Gemini https://ai.google.dev/gemini-api/docs/models
    GPT https://developers.openai.com/api/docs/models
    Cohere https://docs.cohere.com/docs/models
    Kimi https://platform.kimi.ai/docs/models
  4. 在左上角的下拉選單選擇【翻譯設定】，在【翻譯器】欄位選擇你使用的AI名稱。
  5. 若想提高AI翻譯的精準度，請在左上角的下拉選單選擇【LLM設定】，啟用【多輪對話】，並將【對話長度】設定在5以上，長度越高精準度就越好，但Token也會增加。
  6. 按下儲存按鈕儲存剛剛的設定即可開始使用AI翻譯。
