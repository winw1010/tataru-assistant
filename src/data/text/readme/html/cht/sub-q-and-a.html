回首頁

  1. 這程式安全嗎

    本程式不會修改FF14遊戲檔案，且翻譯文字皆使用API請求的方式取得，請放心使用。

  2. 名詞沒有被正確翻譯

    本程式只會修正 對照表 內紀錄的詞彙，若詞彙不在對照表裡將不會被程式修正，請等對照表更新或是使用【回報翻譯錯誤】功能。

  3. 視窗不透明

    AMD和NVIDIA的廣域設定的某些選項會影響視窗的透明度
    請將AMD和NVIDIA的【廣域設定】恢復成預設值即可

  4. 翻譯無法自動顯示下一句

    關閉NVIDIA的Shadowplay即可

  5. 這程式能翻譯什麼語言

    目前僅能翻譯日文和英文版本的遊戲主程式

  6. 能與漢化一起使用嗎

    基本上可以，程式的【翻譯設定】有略過漢化字幕的功能，但請注意本功能無法完美迴避已漢化字幕。

  7. 如果遊戲是日文，隊伍頻道是英文，該如何設定

    至【設定】>【翻譯設定】>將【遊戲語言】設定成日文，【隊伍頻道】設定成英文即可。

  8. 翻譯視窗無法維持在最上層

    請將FF14的顯示設定改成「無邊框全螢幕 / Borderless Window / ボーダレスウィンドウ」。

  9. 每一句都要手動選取翻譯嗎

    不需要 ，一般對話和過場動畫都會自動翻譯，不能自動翻譯的部分也能使用【翻譯螢幕文字】功能來翻譯。

  10. 無法下載翻譯對照表

    若使用加速器或VPN時無法存取Github，請至Tataru Assistant的【設定(齒輪按鈕)】>【系統設定】將SSL驗證關閉。

  11. 出現翻譯失敗的錯誤訊息

    若使用加速器或VPN時無法翻譯字幕，請至Tataru Assistant的【設定(齒輪按鈕)】>【系統設定】將SSL驗證關閉。

  12. 啟動後顯示「這個作業系統不支援 .NET Framework 4.8」的訊息

    請安裝 Microsoft .NET Framework 4.8

  13. 程式無法自動翻譯

    • 請安裝 Microsoft .NET Framework 4.8
    • 至【設定】>【系統設定】，點選【修復字幕讀取器】，修復後重新開機即可
    • 若無法修復請在左下搜尋輸入「cmd」，找到「命令提示字元」對其滑鼠右鍵，點選「以系統管理員身分執行」，然後執行以下指令
      secedit /configure /cfg %windir%\inf\defltbase.inf /db defltbase.sdb /verbose

  14. 如何翻譯對話選項和介面文字

    詳見【翻譯螢幕畫面上的文字】說明頁

  15. 如何重新翻譯

    滑鼠點選程式的翻譯文字，再點選彈出視窗裡的【重新翻譯】按鈕即可

  16. 哪個翻譯引擎比較好

    推薦使用免費流量較多的【Gemini】，其次為【ChatGPT】和【Cohere】，非AI翻譯器的翻譯品質較差不建議使用

  17. 要等很久才會顯示翻譯

    本應用程式是使用API請求翻譯， 因此翻譯速度取決於使用者的網路環境和API伺服器的狀況

  18. 翻譯時跳出錯誤訊息

    表示該翻譯方式目前無法連線，或是已經達到流量限制，只能等待解除限制，或是暫時使用別的翻譯方式

  19. 如何移動視窗

    按住左上的 按鈕即可移動視窗

  20. 旁白字幕無法自動翻譯

    請安裝最新版本

  21. 無法套用變更後的設定

    更改設定後需按下 按鈕才會生效

  22. 如何隱藏按鈕

    開啟【設定】視窗，將【隱藏按鈕】打勾，勾選後滑鼠離開視窗就會自動隱藏按鈕

  23. 如何重置設定

    開啟【設定】視窗，點選【預設】按鈕

  24. 如何清除快取

    開啟【自定翻譯】視窗，點選【清除快取】按鈕

  25. 如何更新

    請至這裡下載最新版本

  26. 如何移除

    在左下方的搜尋輸入【新增或移除程式】，找到本程式並點選移除
