  • Open the [Re-translate & Custom Translation] Window

    Click on the translated text in the Tataru Helper Node

  • Re-translate

    1. Select the [Translation Engine] and [Language]
    2. Click the [Re-translate] button

  • Add Custom Translation

    1. Enter the original text to be replaced in [Before Replacement (Original Text)]
    2. Enter the replacement text in [After Replacement (Custom Translation)]
    3. Click the [Category] menu to select the type of custom translation
    4. Click the [Save Custom Translation] button

  • Delete Custom Translation

    1. Enter the original text of the custom translation to be deleted in [Before Replacement (Original Text)]
    2. Click the [Category] menu to select the category of the custom translation
    3. Click the [Delete Custom Translation] button
