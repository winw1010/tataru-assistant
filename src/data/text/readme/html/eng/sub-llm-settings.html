  1. Click the icon to open Settings, then select [API Settings] from the dropdown menu in the top-left corner.
  2. Apply for an API key from the AI provider's official website (the process is not detailed here). Once you have obtained the API key, enter it into the [API Key] field for that specific AI in the API Settings.
  3. If you wish to use a non-default model, enter the model code you wish to use into the [Model] field for that AI. You can copy model codes from the links below:
    Gemini https://ai.google.dev/gemini-api/docs/models
    GPT https://developers.openai.com/api/docs/models
    Cohere https://docs.cohere.com/docs/models
    Kimi https://platform.kimi.ai/docs/models
  4. In the top-left dropdown menu, select [Translation Settings], and in the [Translator] field, select the AI you are using.
  5. To improve the accuracy of the AI translation, select [LLM Settings] from the top-left dropdown menu, enable [Multi-turn Conversation], and set the [Conversation Length] to 5 or higher. A longer length improves accuracy but will increase token usage.
  6. Click the Save button to store your settings and you can start using the AI translation.
