  1. Is this program safe?

    This program does not modify any Final Fantasy XIV game files. All translations are retrieved via API requests. Please use it with confidence.

  2. Terminology is not being translated correctly

    This program only corrects vocabulary recorded in the reference table. If a term is not in the reference table, the program will not correct it. Please wait for the reference table to be updated or use the [Report Translation Error] feature.

  3. The window is opaque/not transparent

    Certain global settings in AMD and NVIDIA drivers can affect window transparency.
    Please reset your AMD and NVIDIA [Global Settings] to their default values.

  4. Translation does not show the next line automatically

    Please disable NVIDIA Shadowplay.

  5. What languages can this program translate?

    Currently, it can only translate the Japanese and English versions of the game client.

  6. Can it be used with Chinese localization patches?

    Generally, yes. The program's [Translation Settings] include a feature to skip localized subtitles, but please note that this feature cannot perfectly bypass already localized subtitles.

  7. If the game is in Japanese and the party chat is in English, how should I configure it?

    Go to [Settings] > [Translation Settings], set [Game Language] to Japanese, and [Party Chat] to English.

  8. The translation window won't stay on top

    Please change your FFXIV display settings to "Borderless Windowed".

  9. Do I have to manually select the translation for every line?

    No. General dialogue and cutscenes are translated automatically. For parts that cannot be translated automatically, you can use the [Translate Screen Text] function.

  10. Unable to download the translation reference table

    If you are using an accelerator or VPN and cannot access GitHub, please go to Tataru Assistant's [Settings (Gear Icon)] > [System Settings] and turn off SSL verification.

  11. A "Translation Failed" error message appears

    If you are using an accelerator or VPN and cannot translate subtitles, please go to Tataru Assistant's [Settings (Gear Icon)] > [System Settings] and turn off SSL verification.

  12. Upon startup, it displays ".NET Framework 4.8 is not supported by this OS"

    Please install Microsoft .NET Framework 4.8

  13. The program cannot translate automatically

    • Please install Microsoft .NET Framework 4.8.
    • Go to [Settings] > [System Settings] and click [Repair Subtitle Reader], then restart the program.
    • If repair fails, search for "cmd" in the bottom left search bar, right-click "Command Prompt", select "Run as Administrator", and execute the following command:
      secedit /configure /cfg %windir%\inf\defltbase.inf /db defltbase.sdb /verbose

  14. How do I translate dialogue options and interface text?

    See the [Translate Screen Text] explanation page for details.

  15. How do I re-translate?

    Click the translated text in the program with your mouse, then click the [Re-translate] button in the popup window.

  16. Which translation engine is the best?

    [Gemini] is recommended as it offers more free traffic, followed by [ChatGPT] and [Cohere]. Non-AI translators have poorer quality and are not recommended.

  17. It takes a long time for the translation to appear

    This application uses API requests for translation, so the speed depends on your network environment and the status of the API servers.

  18. An error message appears during translation

    This means the current translation method is either unable to connect or has reached its traffic limit. You will need to wait for the restriction to be lifted or use a different translation method temporarily.

  19. How do I move the window?

    Hold down the button in the top left to move the window.

  20. Narrative subtitles are not being translated automatically

    Please install the latest version.

  21. Changes to settings are not being applied

    After changing settings, you must click the button for them to take effect.

  22. How do I hide the buttons?

    Open the [Settings] window and check [Hide Buttons]. Once checked, the buttons will automatically hide when your mouse leaves the window.

  23. How do I reset settings?

    Open the [Settings] window and click the [Default] button.

  24. How do I clear the cache?

    Open the [Custom Translation] window and click the [Clear Cache] button.

  25. How do I update?

    Download the latest version here.

  26. How do I uninstall?

    Search for [Add or remove programs] in the bottom left search bar, find this program, and click uninstall.

  27. How do I check the responses to error reports?

    View them here.
