  • Drag button, hold to move the window

  • Open [Settings] window (Hotkey: Ctrl+F10)

  • Open [ Screen Text Capture ] window, used to translate text that cannot be translated automatically (Hotkey: Ctrl+F11)

  • Enable or disable [Mouse Through] function. When enabled, the mouse can click items behind the Tataru Assistant window.

  • Update notification button, appears automatically when a new version is released.

  • Minimize Tataru Assistant window

  • Close Tataru Assistant window

  • Enable or disable [Text-to-Speech] function. When enabled, text will be converted to Google speech and played.

  • Open [Translation Lookup] window, can be used as a translator. If [Use Tataru Translation] is checked, it will use Tataru Assistant to translate.

  • Open [Read Dialogue History] window to view previous translation records.

  • Delete the last sentence

  • Delete all text
