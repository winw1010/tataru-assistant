  • 使用時機

    遊戲介面和對話選項並不會自動翻譯，需手動選擇擷取

  • 使用方法

    點選 按鈕或是按下Ctrl+F11開啟【翻譯螢幕文字】視窗，然後在擷取視窗中選取欲翻譯的文字(如下方影片所示)

  • 功能說明

    1. 最左邊可選擇擷取方式，有Tesseract OCR和Google Vision兩種
    2. 換行切割：勾選後會以【行】為單位逐行翻譯(用來翻譯過場的選擇項目)，不勾選的話會將所有文字合併成一行再翻譯(翻譯整篇文章時使用)
    3. 編輯文字：勾選後會在擷取文字後顯示編輯視窗，可先在此手動修正文字再按【翻譯】提交，不勾選的話則會直接提交翻譯，若提交的文字有誤也不會修正

  • 擷取方式

    • Tesseract OCR

      大致上沒有問題，但偶爾會有錯字，需手動編輯修正

    • Google Vision

      文字辨識能力較為精準，幾乎不會有錯字，需在GCP建立API憑證才能使用( 查看設定教學 )
