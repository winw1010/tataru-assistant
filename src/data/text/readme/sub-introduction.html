  • 拖曳按鈕，按住可以移動視窗

  • 開啟【設定】視窗，按下Ctrl+F10也能開啟本視窗

  • 開啟【 翻譯螢幕文字 】視窗，用來翻譯無法自動翻譯的文字，按下Ctrl+F11也能開啟本視窗

  • 開啟或關閉【滑鼠穿透】功能，開啟後滑鼠可以點選Tataru Assistant視窗後面的東西

  • 更新提示按鈕，在新版本釋出時會自動出現

  • 縮小Tataru Assistant視窗

  • 關閉Tataru Assistant視窗

  • 開啟或關閉【文字語音】功能，開啟後會將文字轉換成Google語音並播放

  • 開啟【翻譯查詢】視窗，可作為翻譯機使用，若勾選【使用Tataru翻譯】則會使用Tataru Assistant來翻譯

  • 開啟【讀取對話紀錄】視窗，可查看之前的翻譯紀錄

  • 刪除最後一句

  • 刪除全部文字
  • 快捷鍵
    按鍵 功能
    Ctrl+F9 使用說明書
    Ctrl+F10 設定
    Ctrl+F11 翻譯螢幕文字
    Ctrl+F12 開發者工具
