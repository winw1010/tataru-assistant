回首頁
  1. 這程式安全嗎
    此程式無需修改FF14遊戲檔案，且翻譯功能皆使用API請求，請放心使用。
  2. 這程式怎麼運作的
    此程式會擷取遊戲的對話字幕，然後使用API請求翻譯，最後將翻譯結果顯示在程式中。若有勾選【翻譯修正】，則會使用 對照表 修正詞彙。
  3. 名詞沒有被正確翻譯
    此程式只會修正 對照表 內有的詞彙，若詞彙不在對照表裡將不會被程式修正，只能等對照表更新或是以【回報翻譯錯誤】功能來回報。
  4. 視窗無法變透明、翻譯無法自動跳下一句
    由於本程式有使用GPU加速，AMD和NVIDIA的廣域設定的部分選項會讓本程式無法正常運作
    請至AMD和NVIDIA的【廣域設定】將其恢復成預設值即可解決(感謝avbkopye沒牙提供解法)
  5. 這程式支援什麼語言
    支援日文和英文，但翻譯以日文版為主。
  6. 能與漢化一起使用嗎
    基本上可以，程式的【翻譯設定】有略過漢化字幕的功能，但請注意本功能無法完美迴避已漢化字幕。
  7. 如果遊戲是日文，隊伍頻道是英文，該如何設定
    至【設定】>【翻譯設定】>將【遊戲語言】設定成日文，【隊伍頻道】設定成英文即可。
  8. 翻譯視窗無法維持在最上層
    請將FF14的顯示設定改成「無邊框全螢幕 / Borderless Window / ボーダレスウィンドウ」。
  9. 每一句都要手動選取翻譯嗎
    不需要 ，一般對話和過場動畫都會自動翻譯，不能自動翻譯的部分也能使用【翻譯螢幕文字】功能來翻譯。
  10. 無法下載翻譯對照表
    若使用加速器或VPN時無法存取Github，請至Tataru Assistant的【設定(齒輪按鈕)】>【系統設定】將SSL驗證關閉。
  11. 出現翻譯失敗的錯誤訊息
    若使用加速器或VPN時無法翻譯字幕，請至Tataru Assistant的【設定(齒輪按鈕)】>【系統設定】將SSL驗證關閉。
  12. 啟動後顯示「這個作業系統不支援 .NET Framework 4.7.2」的訊息
    請安裝 Microsoft .NET Framework 4.7.2
  13. 程式無法自動翻譯
    • 至【設定】>【系統設定】，點選【修復字幕讀取器】，修復後重新開機即可
    • 若無法修復請在左下搜尋輸入「cmd」，找到「命令提示字元」對其滑鼠右鍵，點選「以系統管理員身分執行」，然後執行以下指令
      secedit /configure /cfg %windir%\inf\defltbase.inf /db defltbase.sdb /verbose
  14. 如何翻譯對話選項和介面文字
    詳見【翻譯螢幕畫面上的文字】說明頁
  15. 如何重新翻譯
    滑鼠點選程式的翻譯文字，再點選彈出視窗裡的【重新翻譯】按鈕即可
  16. 哪個翻譯引擎比較好
    目前推薦使用【Gemini】、【ChatGPT】和【Cohere】等AI翻譯，其次為【有道翻譯】和【彩雲小譯】，但不論哪種都會有翻不好的時候，這時可點選翻譯文字改用其他翻譯引擎重新翻譯
  17. 要等很久才會顯示翻譯
    本應用程式是使用API請求翻譯， 因此翻譯速度取決於使用者的網路環境和API的伺服器狀況
  18. 翻譯時跳出錯誤訊息
    表示該翻譯方式目前無法連線，或是已經達到流量限制，只能等待解除限制，或是暫時使用別的翻譯方式
  19. 如何移動視窗
    按住左上的 按鈕即可移動視窗
  20. 旁白字幕無法自動翻譯
    請安裝最新版本
  21. 無法套用變更後的設定
    更改設定後需按下 按鈕才會生效
  22. 如何隱藏按鈕
    開啟【設定】視窗，將【隱藏按鈕】打勾，勾選後滑鼠離開視窗就會自動隱藏按鈕
  23. 如何重置設定
    開啟【設定】視窗，點選【預設】按鈕
  24. 如何清除快取
    開啟【自定翻譯】視窗，點選【清除快取】按鈕
  25. 如何查看程式錯誤訊息
    Ctrl+F12
  26. 如何更新
    請至這裡下載最新版本
  27. 如何移除
    在左下方的搜尋輸入【新增或移除程式】，找到本程式並點選移除
